
# 🍽️ Zomato Data Analysis Dashboard (Power BI)

## 📊 Overview

This project showcases an interactive **Zomato Data Analysis Dashboard** built using **Power BI**. It provides detailed insights into restaurant distribution, cuisines, ratings, and service availability across multiple countries and cities.

---

## 🚀 Features

* **Global Overview Metrics**

  * Total Restaurants: 9551
  * Total Cuisines: 1826
  * Countries Covered: 15
  * Cities Covered: 141
  * Total Votes: 1M
  * Average Rating: 2.91

* **City-wise Restaurant Analysis**

  * Identifies cities with the highest number of restaurants
  * Helps understand market concentration

* **Restaurant Performance**

  * Count of cuisines offered by each restaurant
  * Average rating comparison across popular brands

* **Cuisine-wise Distribution**

  * Most popular cuisines across regions
  * Helps identify customer preferences

* **Service Availability**

  * 🍽️ Table Booking availability
  * 🚚 Online Delivery availability

* **Interactive Filters**

  * Country selection
  * Year selection

---

## 🛠️ Tools & Technologies

* **Power BI** – Data visualization and dashboard creation
* **Excel / CSV Dataset** – Data source
* **DAX (Data Analysis Expressions)** – Measures and calculations

---

## 📈 Key Insights

* Major cities like **New Delhi** dominate in restaurant count
* North Indian cuisine is the most popular category
* Online delivery is widely available compared to table booking
* Ratings vary significantly across restaurant chains
* A few restaurant brands consistently perform better in ratings
